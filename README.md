# JEMOA
Java Evolutionary Multi-Objective Algorithm
# Data type
* Integer
* Real (double)
* Interval 
# Problems
*  PSP
*  Knapsack
*  DTLZ
*  ZDT
# Algorithms
*  For single-objective:
   * GA: a genetic algorithm. 
*  For multi-objective optimization:
   * GA: A multi-objectve evolutionary algorithm (using dominance)
   * NSGA-II
